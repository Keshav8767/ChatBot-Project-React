# ChatBot Project

A React-based chatbot application built with Vite.

## Features

- Interactive chatbot interface
- Built with React 19
- Fast development with Vite
- Modern UI components

## Installation

```bash
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Technologies

- React 19
- Vite
- ESLint
